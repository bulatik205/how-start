### Начать путь айти стоит с базы. Писать код круто, но сначало надо изучить базу.

![Header](https://capsule-render.vercel.app/api?type=cylinder&height=10&color=gradient)

#### База, что ты должен знать:
1. Компьютер — это железо (процессор, память, диск) и программы. Самая главная программа — операционная система (ОС). Она включает в себя:
- Ядро — управляет процессором, памятью, устройствами
- Интерфейс — то, с чем ты взаимодействуешь (рабочий стол, кнопки, окна)джеры, кнопочки включения/выключеняи ПК, настройки и так далее
2. Железо работает в двоичной системе. Все просто: 0 - нет тока, 1 - есть ток. 
3. Компиляция - пишешь код на C++/C/Rust или другом языке и компилируешь его в машинный код - .exe или другой тип
4. Интерпретация - пишешь код на PHP/Python/JavaScript или другом, а он вполняется по мере нужны интерпретатором.

### Типы данных

![Header](https://capsule-render.vercel.app/api?type=cylinder&height=10&color=gradient)

<div align="start"> 

#### Основные типы данных:
1. Числа: int (целые) и float (дробные)
2. Строки: str — текст в кавычках ("Вася", "привет мир")
3. Логические: bool — только true или false (как выключатель)
4. Символы: char — один символ. В Python отдельно нет, просто строка из одного символа
5. Null/None: null или None — специальное значение, означающее "ничего" или "пусто"
6. Массивы/Списки: array, list:
```
Обычный массив
[1, 2, 3, 4, 5]
```
```
Массив с разными типами данных
["Привет", true, 1, null, 3.14]
```
```
Массив типа ключ-значение (ассоциативный или Hash Map - где как пишут)
[
    "user" => [
        "id" => 10,
        "username" => "Jhon",
        "admin" => true 
    ]
]
```

⚠️ В разных языках синтаксис разный. 

`В PHP используются "[]" и "=>"`

`В Python - "{}" или "[]" и ":" в зависимости от типа массива` 

`В JS - "{}" или "[]" и ":" в зависимости от типа массива`
 
 ⚠️ Индексы. Это условная `ячейка` массива. Исторически сложилось, что индексы считают с 0. Первый элемент в массиве имеет индекс 0. Все остальные имеют `n - 1`, например, 1 элемент считается по формуле `1 - 1` и получаем 0, 10 элемент вычитается также: `10-1` и получаем 9, тоесть, 10 элемент под индексом 9. Если у вас есть массив
 ```
[
    [
        "id" => 10,
        "username" => "Jhon",
        "admin" => true 
    ],
    [
        "id" => 11,
        "username" => "Maria",
        "admin" => false 
    ]
]
 ```
 То чтобы обратиться к вложенному массиву с данными, например, для `Jhon`, то для вывода надо обратиться к `array[0]` и мы получим данные из 
 ```
[
    "id" => 10,
    "username" => "Jhon",
    "admin" => true 
]
 ```
